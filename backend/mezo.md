# mezo reference

**المسار:** `backend`

## وظيفة هذا المكان
الخادم الخلفي للمشروع. مسؤول عن الـ API، قاعدة البيانات عبر Prisma، التوثيق، الطلبات، المنتجات، والسلة.

## المحتويات
- **مجلد `public`**: ملفات عامة يخدمها الباك إند مباشرة عبر Express. تستخدم هنا أساسًا للرفع وعرض الصور.
- **مجلد `src`**: مصدر الباك إند الحقيقي. منه يبدأ السيرفر وتتوزع الملفات بين middleware وroutes وprisma وutils.
- **ملف `.env.example`**: أسماء متغيرات البيئة المطلوبة للباك إند مثل DATABASE_URL وJWT_SECRET وFRONTEND_URL.
- **ملف `package.json`**: اعتماديات وسكربتات الباك إند: dev، build، prisma، seed، وغيرها.
- **ملف `tsconfig.json`**: إعدادات TypeScript الخاصة بالباك إند.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
