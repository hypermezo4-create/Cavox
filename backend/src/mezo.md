# mezo reference

**المسار:** `backend/src`

## وظيفة هذا المكان
مصدر الباك إند الحقيقي. منه يبدأ السيرفر وتتوزع الملفات بين middleware وroutes وprisma وutils.

## المحتويات
- **مجلد `middleware`**: طبقة الحماية والفلترة قبل الوصول إلى الـ routes، مثل التحقق من التوكن أو صلاحية الأدمن.
- **مجلد `prisma`**: تعريف قاعدة البيانات وملفات زرع البيانات الأولية. هذا هو المرجع الرئيسي للداتا موديل.
- **مجلد `routes`**: كل API endpoints الفعلية. كل ملف هنا يمثل مجموعة endpoints تخص جزءًا معينًا من النظام.
- **مجلد `utils`**: دوال مساعدة مشتركة للباك إند، مثل JWT والتحقق وتوليد رقم الطلب.
- **ملف `index.ts`**: نقطة بداية الباك إند. يشغل Express، يربط الـ routes، يفعّل CORS، ويخدم uploads.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
