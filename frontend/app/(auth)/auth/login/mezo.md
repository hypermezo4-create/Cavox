# mezo reference

**المسار:** `frontend/app/(auth)/auth/login`

## وظيفة هذا المكان
صفحة تسجيل الدخول للمستخدم أو الأدمن عبر النظام المربوط مع NextAuth + backend.

## المحتويات
- **ملف `page.tsx`**: واجهة تسجيل الدخول واستدعاء signIn.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
