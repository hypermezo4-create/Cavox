# mezo reference

**المسار:** `frontend/app/(auth)/auth`

## وظيفة هذا المكان
مجلد auth الفعلي. يحتوي صفحات login وregister الظاهرة للمستخدم.

## المحتويات
- **مجلد `login`**: صفحة تسجيل الدخول للمستخدم أو الأدمن عبر النظام المربوط مع NextAuth + backend.
- **مجلد `register`**: صفحة إنشاء حساب جديد ثم تسجيل دخول تلقائي بعد نجاح التسجيل.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
