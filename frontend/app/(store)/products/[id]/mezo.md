# mezo reference

**المسار:** `frontend/app/(store)/products/[id]`

## وظيفة هذا المكان
صفحة تفاصيل المنتج الواحد مع الصور والخيارات والمراجعات.

## المحتويات
- **ملف `page.tsx`**: صفحة المنتج الواحد. غالبًا تتعامل مع slug أو id وتعرض التفاصيل.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
