# mezo reference

**المسار:** `frontend/app/api/chat`

## وظيفة هذا المكان
واجهة شات من جهة الفرونت إن احتجت proxy أو endpoint محلي للواجهة.

## المحتويات
- **ملف `route.ts`**: Endpoint محلي للشات من الواجهة إن كانت الواجهة تمرر الطلب منه.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
