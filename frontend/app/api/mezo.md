# mezo reference

**المسار:** `frontend/app/api`

## وظيفة هذا المكان
API routes خاصة بالفرونت/Next.js وليست Express backend. تستخدم هنا للمصادقة والشات.

## المحتويات
- **مجلد `auth`**: المسار الخاص بـ NextAuth في الواجهة.
- **مجلد `chat`**: واجهة شات من جهة الفرونت إن احتجت proxy أو endpoint محلي للواجهة.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
