# mezo reference

**المسار:** `frontend/app/(store)`

## وظيفة هذا المكان
مجموعة صفحات الواجهة العامة للمتجر: الرئيسية، الأقسام، المنتجات، السلة، الشيك أوت، وغيرها.

## المحتويات
- **مجلد `about`**: صفحة من نحن/حول Cavo.
- **مجلد `cart`**: صفحة السلة. تقرأ محتويات الكارت من الباك إند وتسمح بتعديل الكمية أو الحذف.
- **مجلد `checkout`**: صفحة إتمام الشراء. تجمع بيانات العميل ثم ترسل order فعلي للباك إند.
- **مجلد `contact`**: صفحة التواصل مع نموذج إرسال الرسائل وروابط التواصل.
- **مجلد `kids`**: صفحة قسم الأطفال. تعرض منتجات category الأطفال.
- **مجلد `men`**: صفحة قسم الرجال.
- **مجلد `offers`**: صفحة العروض والمنتجات المخفضة أو البانرات ذات الصلة.
- **مجلد `order-confirmation`**: مجموعة مسارات تأكيد الطلب بعد الشراء.
- **مجلد `privacy-policy`**: صفحة سياسة الخصوصية.
- **مجلد `products`**: مسارات صفحات المنتجات العامة: القائمة وتفاصيل المنتج.
- **مجلد `shop`**: صفحة تسوق عامة بديلة لقائمة المنتجات أو landing للكتالوج.
- **مجلد `terms-of-service`**: صفحة الشروط والأحكام.
- **مجلد `women`**: صفحة قسم النساء.
- **ملف `page.tsx`**: الصفحة الرئيسية للمتجر.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
