# mezo reference

**المسار:** `frontend/components/admin`

## وظيفة هذا المكان
مكونات مساعدة لصفحات لوحة الإدارة مثل shells والنماذج ورسائل الحالة.

## المحتويات
- **ملف `AdminPageShell.tsx`**: Wrapper موحد لصفحات الأدمن: عنوان، وصف، مساحات وأزرار.
- **ملف `ProductForm.tsx`**: النموذج الرئيسي لإنشاء وتعديل المنتجات داخل الأدمن.
- **ملف `StatusMessage.tsx`**: رسالة حالة موحدة للنجاح/الخطأ في صفحات الأدمن.
- **ملف `UploadButton.tsx`**: زر رفع الصور وربطه مع API الرفع.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
