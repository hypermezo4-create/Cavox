# mezo reference

**المسار:** `frontend/components/ui`

## وظيفة هذا المكان
مكونات UI صغيرة قابلة لإعادة الاستخدام مثل الأزرار والـ badges.

## المحتويات
- **ملف `Badge.tsx`**: شارات UI صغيرة مثل New أو Sale.
- **ملف `PremiumButton.tsx`**: زر موحد بتصميم Cavo.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
