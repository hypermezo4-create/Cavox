# mezo reference

**المسار:** `frontend/lib`

## وظيفة هذا المكان
طبقة utilities والـ API clients والربط مع الباك إند.

## ملاحظة مهمة
حاول إبقاء كل calls إلى backend هنا بدل توزيعها عشوائيًا داخل الصفحات.

## المحتويات
- **ملف `admin-api.ts`**: طبقة استدعاءات الأدمن للباك إند بشكل منظم.
- **ملف `api.ts`**: طبقة استدعاءات الواجهة العامة للباك إند.
- **ملف `auth.ts`**: مساعدات خاصة بالتوثيق وربط الواجهة بالمستخدم الحالي.
- **ملف `brand.ts`**: ثوابت أو بيانات تخص هوية Cavo ونصوصها/ألوانها.
- **ملف `i18n.ts`**: مساعدات اللغة أو القواميس والاتجاه.
- **ملف `utils.ts`**: دوال مساعدة عامة للفرونت.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
