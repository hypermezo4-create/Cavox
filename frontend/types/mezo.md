# mezo reference

**المسار:** `frontend/types`

## وظيفة هذا المكان
تعريفات TypeScript الإضافية أو توسيعات مكتبات مثل NextAuth.

## المحتويات
- **ملف `next-auth.d.ts`**: توسيع Types الخاصة بـ NextAuth لإضافة user fields مثل token/role.

## كيف أتعامل معه لاحقًا
استخدم هذا الملف كمرجع سريع قبل التعديل. لو ستغير منطقًا هنا، راجع الملفات المرتبطة المذكورة أعلاه حتى لا يحدث كسر في جزء آخر من المشروع.
